### Endpointlar:

| Endpoint    | Vazifasi                     | Method |
| ----------- | ---------------------------- | ------ |
| `/todo`     | barcha todolarni qaytaradi   | GET    |
| `/todo/:id` | id bo'yicha todo qaytaradi   | GET    |
| `/todo/:id` | id bo'yicha todoni o'chiradi | DELETE |
